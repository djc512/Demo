本协议为《印家打印服务协议》修订版本，自本协议发布之日起，印家打印平台各处所称“印家打印服务协议”均指本协议。

【签约动作】当您按照注册页面提示填写信息、阅读并同意本协议且完成全部注册程序后，即表示您已充分阅读、理解并接受本协议的全部内容，并与印家打印达成一致，成为印家打印平台“用户”。阅读本协议的过程中，如果您不同意本协议或其中任何条款约定，您应立即停止注册程序。

 

 

 

一、 定义

 

 

 

印家打印平台：

 

 

印家打印平台服务：印家打印基于互联网，以包含印家打印平台网站、客户端等在内的各种形态（包括未来技术发展出现的新的服务形态）向您提供的各项服务。

 

 

 

印家打印平台规则：包括在所有印家打印平台规则频道内已经发布及后续发布的全部规则、解读、公告等内容以及各平台在帮派、论坛、帮助中心内发布的各类规则、实施细则、产品流程说明、公告等。

 

 

 

印家打印平台规则：印家打印平台规则频道列明的各项规则

 

 

 

同一用户：使用同一身份认证信息或经印家打印排查认定多个印家打印账户的实际控制人为同一人的，均视为同一用户。

 

 

 

二、 协议范围

 

 

 

2.1 签约主体

 

 

 

【平等主体】本协议由您与印家打印平台经营者共同缔结，本协议对您与印家打印平台经营者均具有合同效力。

 

 

 

【主体信息】印家打印平台经营者是指经营印家打印平台的各法律主体，您可随时查看印家打印平台各网站首页底部公示的证照信息以确定与您履约的印家打印主体。本协议项下，印家打印平台经营者可能根据印家打印平台的业务调整而发生变更，变更后的印家打印平台经营者与您共同履行本协议并向您提供服务，印家打印平台经营者的变更不会影响您本协议项下的权益。印家打印平台经营者还有可能因为提供新的印家打印平台服务而新增，如您使用新增的印家打印平台服务的，视为您同意新增的印家打印平台经营者与您共同履行本协议。发生争议时，您可根据您具体使用的服务及对您权益产生影响的具体行为对象确定与您履约的主体及争议相对方。

 

 

 

2.2补充协议

 

 

 

由于互联网高速发展，您与印家打印签署的本协议列明的条款并不能完整罗列并覆盖您与印家打印所有权利与义务，现有的约定也不能保证完全符合未来发展的需求。因此，印家打印平台法律声明及隐私权政策、印家打印平台规则均为本协议的补充协议，与本协议不可分割且具有同等法律效力。如您使用印家打印平台服务，视为您同意上述补充协议。

 

 

 

三、 账户注册与使用

 

 

 

3.1 用户资格

 

 

 

您确认，在您开始注册程序使用印家打印平台服务前，您应当具备中华人民共和国法律规定的与您行为相适应的民事行为能力。若您不具备前述与您行为相适应的民事行为能力，则您及您的监护人应依照法律规定承担因此而导致的一切后果。

 

此外，您还需确保您不是任何国家、国际组织或者地域实施的贸易限制、制裁或其他法律、规则限制的对象，否则您可能无法正常注册及使用印家打印平台服务。

 

 

 

3.2 账户说明

 

 

 

【账户获得】当您按照注册页面提示填写信息、阅读并同意本协议且完成全部注册程序后，您可获得印家打印平台账户并成为印家打印平台用户。

 

印家打印平台只允许每位用户使用一个印家打印平台账户。如有证据证明或印家打印平台有理由相信您存在不当注册或不当使用多个印家打印平台账户的情形，印家打印平台可采取冻结或关闭账户、取消订单、拒绝提供服务等措施，如给印家打印平台及相关方造成损失的，您还应承担赔偿责任。

 

 

 

【账户使用】您有权使用您设置或确认的印家打印会员名、邮箱、手机号码（以下简称“账户名称”）及您设置的密码（账户名称及密码合称“账户”）登录印家打印平台。

 

您有权使用您的印家打印账户登录阿里平台而无需另行设置账户，但您的印家打印账户于2013年3月18日前注册或者您注册印家打印账户时所使用的邮箱或手机号码已在阿里平台注册或使用过的除外。

 

由于您的印家打印账户关联您的个人信息及印家打印平台商业信息，您的印家打印账户仅限您本人使用。未经印家打印平台同意，您直接或间接授权第三方使用您印家打印账户或获取您账户项下信息的行为无效。如印家打印平台判断您印家打印账户的使用可能危及您的账户安全及/或印家打印平台信息安全的，印家打印平台可拒绝提供相应服务或终止本协议。

 

 

 

3.3 注册信息管理

 

 

 

【会员名的合法性】您设置的印家打印会员名不得违反国家法律法规及印家打印平台规则关于会员名的管理规定，否则印家打印可回收您的印家打印会员名。印家打印会员名的回收不影响您以邮箱、手机号码登录印家打印平台并使用印家打印平台服务。

 

 

 

3.4 账户安全规范

 

 

 

【账户安全保管义务】您的账户为您自行设置并由您保管，印家打印任何时候均不会主动要求您提供您的账户密码。因此，建议您务必保管好您的账户，并确保您在每个上网时段结束时退出登录并以正确步骤离开印家打印平台。

 

账户因您主动泄露或因您遭受他人攻击、诈骗等行为导致的损失及后果，印家打印并不承担责任，您应通过司法、行政等救济途径向侵权行为人追偿。

 

 

 

【账户行为责任自负】除印家打印存在过错外，您应对您账户项下的所有行为结果（包括但不限于在线签署各类协议、发布信息、购买商品及服务及披露信息等）负责。

 

 

 

【日常维护须知】如发现任何未经授权使用您账户登录印家打印平台或其他可能导致您账户遭窃、遗失的情况，建议您立即通知印家打印，您理解印家打印对您的任何请求采取行动均需要合理时间，且印家打印应您请求而采取的行动可能无法避免或阻止侵害后果的形成或扩大，除印家打印存在法定过错外，印家打印不承担责任。

 

 

 

4.5费用

 

 

 

印家打印为印家打印平台向您提供的服务付出了大量的成本，除印家打印平台明示的收费业务外，印家打印向您提供的服务目前是免费的。如未来印家打印向您收取合理费用，印家打印会采取合理途径并以足够合理的期限提前通过法定程序并以本协议第八条约定的方式通知您，确保您有充分选择的权利。

印家打印非常重视用户个人信息（即能够独立或与其他信息结合后识别用户身份的信息）的保护，在您使用印家打印提供的服务时，您同意印家打印按照在印家打印平台上公布的隐私权政策收集、存储、使用、披露和保护您的个人信息。印家打印希望通过隐私权政策向您清楚地介绍印家打印对您个人信息的处理方式，因此印家打印建议您完整地阅读隐私权政策（点击此处或点击印家打印平台首页底部链接），以帮助您更好地保护您的隐私权。

 

 

 

 

六、 用户的违约及处理

 

 

 

6.1 违约认定

 

 

 

发生如下情形之一的，视为您违约：

 

（一）使用印家打印平台服务时违反有关法律法规规定的；

 

（二）违反本协议或本协议补充协议（即本协议第2.2条）约定的。

 

为适应电子商务发展和满足海量用户对高效优质服务的需求，您理解并同意，印家打印可在印家打印平台规则中约定违约认定的程序和标准。如：印家打印可依据您的用户数据与海量用户数据的关系来认定您是否构成违约；您有义务对您的数据异常现象进行充分举证和合理解释，否则将被认定为违约。

 

 

 

 

七、 协议的变更

 

 

 

印家打印可根据国家法律法规变化及维护交易秩序、保护消费者权益需要，不时修改本协议、补充协议，变更后的协议、补充协议（下称“变更事项”）将通过法定程序并以本协议第八条约定的方式通知您。

 

如您不同意变更事项，您有权于变更事项确定的生效日前联系印家打印反馈意见。如反馈意见得以采纳，印家打印将酌情调整变更事项。

 

如您对已生效的变更事项仍不同意的，您应当于变更事项确定的生效之日起停止使用印家打印平台服务，变更事项对您不产生效力；如您在变更事项生效后仍继续使用印家打印平台服务，则视为您同意已生效的变更事项。

 

 

 

八、 通知

 

 

 

8.1有效联系方式

 

您在注册成为印家打印平台用户，并接受印家打印平台服务时，您应该向印家打印提供真实有效的联系方式（包括您的电子邮件地址、联系电话、联系地址等），对于联系方式发生变更的，您有义务及时更新有关信息，并保持可被联系的状态。

 

您在注册印家打印平台用户时生成的用于登陆印家打印平台接收站内信、系统消息和阿里旺旺即时信息的会员账号（包括子账号），也作为您的有效联系方式。

 

印家打印将向您的上述联系方式的其中之一或其中若干向您送达各类通知，而此类通知的内容可能对您的权利义务产生重大的有利或不利影响，请您务必及时关注。

 

您有权通过您注册时填写的手机号码或者电子邮箱获取您感兴趣的商品广告信息、促销优惠等商业性信息；您如果不愿意接收此类信息，您有权通过印家打印提供的相应的退订功能进行退订。

 

 

 

8.2 通知的送达

 

印家打印通过上述联系方式向您发出通知，其中以电子的方式发出的书面通知，包括但不限于在印家打印平台公告，向您提供的联系电话发送手机短信，向您提供的电子邮件地址发送电子邮件，向您的账号发送旺旺信息、系统消息以及站内信信息，在发送成功后即视为送达；以纸质载体发出的书面通知，按照提供联系地址交邮后的第五个自然日即视为送达。

 

对于在印家打印平台上因交易活动引起的任何纠纷，您同意司法机关（包括但不限于人民法院）可以通过手机短信、电子邮件或阿里旺旺等现代通讯方式或邮寄方式向您送达法律文书（包括但不限于诉讼文书）。您指定接收法律文书的手机号码、电子邮箱或阿里旺旺账号等联系方式为您在印家打印平台注册、更新时提供的手机号码、电子邮箱联系方式以及在注册印家打印用户时生成阿里旺旺账号，司法机关向上述联系方式发出法律文书即视为送达。您指定的邮寄地址为您的法定联系地址或您提供的有效联系地址。

 

您同意司法机关可采取以上一种或多种送达方式向您达法律文书，司法机关采取多种方式向您送达法律文书，送达时间以上述送达方式中最先送达的为准。

 

您同意上述送达方式适用于各个司法程序阶段。如进入诉讼程序的，包括但不限于一审、二审、再审、执行以及督促程序等。

 

你应当保证所提供的联系方式是准确、有效的，并进行实时更新。如果因提供的联系方式不确切，或不及时告知变更后的联系方式，使法律文书无法送达或未及时送达，由您自行承担由此可能产生的法律后果。

 

 

 

九、 协议的终止

 

 

 

9.1 终止的情形

 

 

 

【用户发起的终止】您有权通过以下任一方式终止本协议：

 

（一）在满足印家打印平台公示的账户注销条件时您通过网站自助服务注销您的账户的；

 

（二）变更事项生效前您停止使用并明示不愿接受变更事项的；

 

（三）您明示不愿继续使用印家打印平台服务，且符合印家打印平台终止条件的。

 

 

 

【印家打印发起的终止】出现以下情况时，印家打印可以本协议第八条的所列的方式通知您终止本协议：

 

（一）您违反本协议约定，印家打印依据违约条款终止本协议的；

 

（二）您盗用他人账户、发布违禁信息、骗取他人财物、售假、扰乱市场秩序、采取不正当手段谋利等行为，印家打印依据印家打印平台规则对您的账户予以查封的；

 

（三）除上述情形外，因您多次违反印家打印平台规则相关规定且情节严重，印家打印依据印家打印平台规则对您的账户予以查封的；

 

（四）您的账户被印家打印依据本协议回收的；

 

（五）其它应当终止服务的情况。

 

 

 

9.2 协议终止后的处理

 

 

 

【用户信息披露】本协议终止后，除法律有明确规定外，印家打印无义务向您或您指定的第三方披露您账户中的任何信息。

 

 

 

【印家打印权利】本协议终止后，印家打印仍享有下列权利：

 

（一）继续保存您留存于印家打印平台的本协议第五条所列的各类信息；

 

（二）对于您过往的违约行为，印家打印仍可依据本协议向您追究违约责任。

 

 

 

【交易处理】本协议终止后，对于您在本协议存续期间产生的交易订单，印家打印可通知交易相对方并根据交易相对方的意愿决定是否关闭该等交易订单；如交易相对方要求继续履行的，则您应当就该等交易订单继续履行本协议及交易订单的约定，并承担因此产生的任何损失或增加的任何费用。

【可分性】本协议任一条款被视为废止、无效或不可执行，该条应视为可分的且并不影响本协议其余条款的有效性及可执行性。
